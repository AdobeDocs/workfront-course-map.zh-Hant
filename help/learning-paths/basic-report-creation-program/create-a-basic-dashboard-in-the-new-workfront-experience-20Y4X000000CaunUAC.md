# 建立基本儀表板

以下四個 Workfront One 學習路徑，

* 建立基本儀表板
* 建立基本自訂報告
* 報告設定
* 傳送和共用報告

已合併為單一 Experience League 課程「[建立自訂報告和儀表板](https://experienceleague.adobe.com/?recommended=Workfront-U-1-2022.3.reporting){target="_blank"}」。
