# 新 Workfront 體驗中的請求

*新 Workfront 體驗中的請求* Workfront One 學習路徑已移轉為 Experience League 上的「[使用者的請求佇列](https://experienceleague.adobe.com/?recommended=Workfront-U-1-2022.2.request-queues){target="_blank"}」
