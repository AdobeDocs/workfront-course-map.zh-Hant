# 新 Workfront 體驗的建立基本報告

報告可提供分析、激勵行動並證明工作成果。瞭解如何建立令人驚嘆且富有洞察力的自訂報告和儀表板，以釋放您在 Workfront 中追蹤的資料。

>[!IMPORTANT]
>
>**請注意，先前的 Workfront One 學習計畫現已對應到剛移轉並最佳化的 Experience League Workfront 課程。所有相同的學習目標皆已納入新結構中**。
